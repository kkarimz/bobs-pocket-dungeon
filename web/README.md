# Bob's Pocket Dungeon (Web)

Interactive solo crawl — Vite + React + TypeScript, static hosting.

```bash
cd web
npm install
npm run dev      # local play
npm run build    # output → dist/
npm run preview  # serve dist locally
```

## GitHub Pages

Pushes to `main` deploy via GitHub Actions.

**Live:** https://kkarimz.github.io/bobs-pocket-dungeon/

Local preview with the Pages base path:

```bash
cd web && npm run build && npm run preview
```
